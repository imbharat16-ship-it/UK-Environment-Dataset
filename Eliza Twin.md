The actual transcript (`857315_data/Households11_20/Household12Eliza/20220325_Final_interview_Eliza.docx`) has **115 substantive interviewer turns** (131 total `A:` lines, many are short follow-ups like "A trial?" or "Like this?").

For a twin, the goal is **not** to reproduce whether Eliza remembers "on Tuesday I ate quiche in a plastic box." That is fact recall / lived experience.

The goal **is** to reproduce **how she behaves** when a situation arises:


| Situation type                 | Behavioral pattern to simulate                                                                                         |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| End-of-term time pressure      | Skip proper meals; default to biscuits/ramen/pre-packaged pasta; batch-freeze when she does cook                       |
| Student budget vs green choice | Bulk chicken; Aldi shop; won't buy £2 containers; airport meal deal outbound, homemade sandwich inbound                |
| Recycling decision             | Thick/rigid plastics + green arrow → recycle (rinse chicken tray); thin films / party cups / ambiguous → general waste |
| Social learning                | Pick up flatmate habits (freezer bags, clingfilm fridge covers, batch freezing)                                        |
| Food waste                     | Chain leftovers (fajita sauce → noodles); mum-recipe logic ("use everything"); freeze surplus                          |
| Occasion shift                 | Kitchen = more diligence; parties/clubbing = one bin bag, no sorting                                                   |
| Attitude–behavior gap          | Knows plastic harms environment; lacks scientific detail; budget/convenience wins in practice                          |


**Tagging logic:** Each I2 turn gets tagged with **which behavioral patterns from prior steps should activate** — not which diary photo to cite.

### Question type legend

Each turn is also classified by what the twin must do:


| Type  | Meaning                   | Twin approach                                                                                 |
| ----- | ------------------------- | --------------------------------------------------------------------------------------------- |
| **B** | Behavior probe            | Simulate decision logic from calibrated patterns                                              |
| **F** | Fact recall               | Weak for twin — interviewer asks "can you remember Tuesday" or "what brand"                   |
| **P** | Photo elicitation         | Needs photo context OR generalize to food-type behavior                                       |
| **C** | Clarification / follow-up | Continues prior turn; inherits parent tags                                                    |
| **R** | Reflexive look-back       | Interviewer names prior step; twin should **re-express known patterns**, not invent new facts |
